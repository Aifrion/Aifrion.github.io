HRI: Jack Bot

An embodied party-game host, built on the Mini Pupper quadruped.

Demo: Jack Bot hosting a round — speech, player tracking, and elimination walk-up.

CSE 276B · Human-Robot Interaction · Dr. Laurel Riek, UC San Diego
By Brian Yee, Ashley Shin, and Sherry Tseng

Jackbox-style party games put the game master on a screen. Jack Bot puts it in the room. Following the "phygital" play literature, our team turned the Mini Pupper into an embodied game host for a Mafia-style party game: it narrates with text-to-speech (Piper-TTS), finds players by shirt color with OpenCV color-thresholding, and physically walks up to whoever gets eliminated.

Machine learning for health, at UC Irvine.

Identifying Sleep Behavior Subtypes Among Dementia Caregivers (Formal Experience)

Unsupervised clustering on wearable sensor data · UC Irvine · unpublished

Dementia caregivers carry an enormous, often invisible burden — and it shows up in how they sleep. Using wearable sensor data, this work applied unsupervised clustering to uncover distinct sleep behavior subtypes among caregivers.

  • Aligned sleep data preprocessing and feature selection with the standardized Ru-SATED sleep health framework, keeping the analysis robust and theoretically grounded.
  • Engineered key metrics for sleep pattern analysis — including daily and aggregate sleep midpoint calculations — to quantify sleep regularity and timing variability for clustering.
  • Unsupervised Learning
  • Wearable Sensors
  • Digital Health
  • Feature Engineering

Deep learning · computer vision · independent project
